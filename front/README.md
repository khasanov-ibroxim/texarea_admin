# Blog Admin Panel

Next.js 14 admin panel - session authentication bilan.

## Ishga Tushirish

### 1. Dependencies
```bash
npm install
```

### 2. Backend ishga tushirish
Backend `http://localhost:5000` da ishlashi kerak!

### 3. Start
```bash
npm run dev
```

URL: http://localhost:3001
